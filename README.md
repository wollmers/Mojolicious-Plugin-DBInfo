# SYNOPSIS

    $app->plugin('Mojolicious::Plugin::DBInfo');

# DESCRIPTION

[Mojolicious::Plugin::DBInfo](https://metacpan.org/pod/Mojolicious::Plugin::DBInfo) is a Mojolicious-Plugin.

It creates a route

    /dbinfo
    

where detailed info is dispalyed in formatted HTML.
